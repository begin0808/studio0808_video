Recording Assistant

01. Overview

Recording Assistant is a lightweight tool with live waveform display and AI denoise, perfect for high-quality voice samples.

02. Core Features

🎤 Device Selection

  • 👉 Vocal Recording: Select Physical Microphone (e.g. Realtek Audio).
  • 👉 Internal Recording: Select Stereo Mix to capture system sounds.

📋 Common Devices

Device Status Description
Microsoft Sound Mapper ✅ OK Points to Windows default devices.
Microphone (Realtek / USB Audio) ⭐ Best Best for vocal recording.
Stereo Mix 🎧 Int. Records internal PC audio.

03. Steps

  1. Record: Click "🔴 Record" to start.
  2. Review: Click "▶ Play" to check quality.
  3. File: Open the output folder to find your .wav.
